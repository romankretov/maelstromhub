# Roadmap

## Phase 0: Foundation

- Monorepo scaffold.
- FastAPI, Next.js, worker, and shared core package.
- Docker Compose for local infrastructure.
- Product, architecture, roadmap, and coding standards documentation.

## Phase 1: Research Workspace

- Market data ingestion design.
- Historical data storage model.
- Research run lifecycle.
- Experiment metadata and result persistence.
- Web screens for browsing runs and datasets.

## Phase 2: Backtesting

- Strategy interface.
- Deterministic backtest runner.
- Fee, funding, slippage, and latency assumptions.
- Result comparison and reporting.
- Regression fixtures for strategy behavior.

## Phase 3: Operations Readiness

- Job queue implementation.
- Structured logging and tracing.
- Metrics and health dashboards.
- Role-aware operational workflows.
- Deployment plan.

## Phase 4: Trading Design

- Risk model.
- Permission and key-management plan.
- Dry-run execution engine.
- Kill switches and operator approvals.
- Audit trail.

Live trading remains blocked until Phase 4 is complete and reviewed.
